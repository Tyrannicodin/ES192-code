# NxN Resistor Array
To run the NxN resistor array:
- Compile `arduino.ino` and upload it to an Arduino Uno R4, connected to the sensor as shown in the report.
- Connect the Arduino to the PC and find its serial port.
- Open `matlab.m` in matlab, and set the `port` variable to the serial port.
- Run the matlab code. A graph will be displayed on the PC.
